# Loop Catalog

This workspace uses a deterministic SDLC loop:

1. `BOOTSTRAP`
2. `UNDERSTAND`
3. `DISCOVER`
4. `ANALYZE`
5. `PLAN`
6. `DESIGN`
7. `RISK REVIEW`
8. `IMPLEMENT`
9. `SELF REVIEW`
10. `VERIFY`
11. `SECURITY REVIEW`
12. `PERFORMANCE REVIEW`
13. `DOCUMENT`
14. `INTEGRATE`
15. `UPDATE MEMORY`
16. `RETROSPECTIVE`
17. `DONE`

The loop is verifier-led. If a verifier fails, return to the earliest affected state and make the smallest safe correction.
