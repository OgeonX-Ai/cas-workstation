## Milestone Summary

**Key decisions:**
- Coverage gates ratchet from measured baselines — never hardcode aspirational 100% (deviation recorded openly).
- Registry canonical namespace moved to the live Pages URL (BREAKING, documented) over resurrecting the dead custom domain.
- NO-AZURE **deploy** lock: bicep authoring/linting allowed, deployment forbidden until deliberately lifted.
- Agent governance codified: PR-only sub-repo changes, no self-approval (enforced and honored), one agent per working tree, model tiering (opus/sonnet/haiku by failure-mode), delegation depth as policy.
- Every failure class found in execution became an automated check the same week (encoding hazard, stale tracking refs, untruthful commits, WSL worktrees, unpushed work).

**Recurring lesson:** parallel AI sessions strand uncommitted/unpushed work — three separate incidents caught and rescued; detection now automated.
