# Phase 7 Context

Pilot evidence is committed, deterministic JSON with a strict schema and a Windows-native validator. Each scenario records bounded events, assertions, artifact URIs, source commits, and exact reproduction commands. Evidence generation and validation are local-only and perform no push, deployment, message, or production mutation.
