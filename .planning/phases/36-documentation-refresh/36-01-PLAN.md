at the time of writing (get via `git -C portfolio/<repo> rev-parse origin/<default-branch>`
or local HEAD if already synced to default branch). `<date>` = ISO date (YYYY-MM-DD) of
the writing session. This is advisory metadata for the Phase 34 staleness heuristic
(36-04-PLAN.md) — not a build artifact, so it is fine for it to go stale until the next
refresh.
