<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| PR branch -> origin/main | New branch pushed and PR opened but never merged; branch protection and default-branch content untouched |
| Self-hosted Windows runner (fixer.yml) -> host machine | Unlike GitHub-hosted runners, a self-hosted runner's compromise persists on the physical host; its token scope is the primary blast-radius control |
| CI coverage gate -> merge decision | --cov-fail-under is advisory only until this PR is merged (out of scope); it does not yet block anything in production until Phase 30's merge-train equivalent lands this PR |
