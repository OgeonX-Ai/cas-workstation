<objective>
Remediate GitHub Actions supply-chain and least-privilege gaps (S1, S2, S3 from
docs/improvement-backlog.md; REQ-1.4.10) in four portfolio repos: Promptimprover, autogen,
autopilot-core, autopilot-demo.

Purpose: Pin every third-party action to an immutable commit SHA (closing the supply-chain
tampering window a mutable tag like `@v4` leaves open) and ensure every workflow declares
least-privilege `permissions:` and a `timeout-minutes:` bound on every job, so a compromised or
hung workflow cannot silently escalate token scope or run indefinitely.

Output: Updated workflow YAML in 4 repos, each pushed to a new branch off `origin/main` (or
`origin/master` for Promptimprover) and opened as a PR (no merge — merges are blocked by
environment policy). autopilot-demo is audited and confirmed already-compliant with no PR
needed.
